# Notas al pie en [[Markdown]] usando #Obsidian

This is a simple footnote[^1]. En este texto se inserta otra nota al pie, por ejemplo, esta[^2]

---

[^1]: This is the referenced text.
[^2]: Add 2 spaces at the start of each new line. This lets you write footnotes that span multiple lines.
[^note]: Named footnotes still appear as numbers, but can make it easier to identify and link references.

## Comentarios

Los comentarios se hacen usando Doble signo de porcentaje (como en Latex, un poco), al principio y final del comentario

%%(si se exporta, estas líneas no las leerá)%%
\* Se agrega la barra a la izquierda para no ejecutar la función de cada símbolo

# por cierto!!!

Las notas se etiquetan usando un asterisco seguido del nombre de la etiqueta y así es más fácil encontrar las notas
